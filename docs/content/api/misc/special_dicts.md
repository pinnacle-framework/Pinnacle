## `diff` 

```python
diff(r1,
     r2)
```
| Parameter | Description |
|-----------|-------------|
| r1 | Dict |
| r2 | Dict |

Get the difference between two dictionaries.

```python
_diff({'a': 1, 'b': 2}, {'a': 2, 'b': 2})
# {'a': (1, 2)}
_diff({'a': {'c': 3}, 'b': 2}, {'a': 2, 'b': 2})
# {'a': ({'c': 3}, 2)}
```

# Example:
